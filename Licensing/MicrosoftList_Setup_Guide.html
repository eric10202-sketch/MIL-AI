Part 1 – Create the Microsoft List

1Open Microsoft Lists

Go to lists.microsoft.com or open it from Microsoft 365 App Launcher → Lists.
2Create a new list from CSV

  1. Click "New list"
  2. Select "From CSV"
  3. Upload Software_License_MicrosoftList_Import.csv (located in your project folder)
  4. Microsoft Lists will show a preview – verify columns look correct
  5. Name the list: Software License Database
  6. Click Create
⚠ If your SharePoint site is the preferred location, choose "In a team site" and select your site before uploading.

How to change a column type

  1. Click the column header → Column settings → Edit
  2. Change Type to the recommended type above
  3. Save

Part 3 – Recommended Views

Create the following views in Microsoft Lists for easier management:

View NameFilterGroup BySort
All Licenses(none)Source_ProjectVendor → Title
ASTRA – Paid LicensesSource_Project = ASTRA AND License_Category = Paid LicenseVendorItem_No
ASTRA – Freeware & OSSSource_Project = ASTRA AND License_Category ≠ Paid LicenseVendorItem_No
FRAME – AllSource_Project = FRAMEVendorItem_No
FRAME – Pending ProcurementSource_Project = FRAME AND Ready_for_Procurement is not emptyResponsibleDue_Date

How to create a view

  1. In your list, click + Add view (top right)
  2. Give it a name, set filters, grouping, and sort
  3. Save

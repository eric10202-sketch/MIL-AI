# Repository Governance Updates

## Purpose
Maintain repository-level documentation hygiene and traceability.

## Scope
- CLAUDE.md repository reference tables
- Project folder inventory
- Generator scripts inventory
- Active project files inventory
- Last reviewed date updates

## Rules
- Update metadata whenever new project folder or generator script is added.
- Keep active/closed project status current.
- Keep naming conventions consistent across docs and generated outputs.
- Record methodology changes in the appropriate concept or benchmark section.
